fitzRoy v1.7.0 on CRAN

Author

James Day

Published

March 12, 2025

I’m pleased to announce that fitzRoy v1.7.0 has just been accepted to CRAN!

You can read about all the changes below.

library(fitzRoy)

New Features

Fantasy and Supercoach scores

New functions have been added to fetch AFL Fantasy (Dream Team) and Supercoach round scores from Footywire.

  • fetch_scores() - Wrapper function to fetch either scoring system. Specify type = "supercoach" or type = "dream_team" along with a year and rounds.
  • fetch_supercoach_scores() - Fetch Supercoach scores directly.
  • fetch_fantasy_scores() - Fetch AFL Fantasy scores directly.

AFL Awards

A new fetch_awards() wrapper function has been added to retrieve data for three major AFL awards from Footywire.

  • fetch_awards_brownlow() - Returns player and team Brownlow vote tallies for a given season.
  • fetch_awards_allaustralian() - Returns the final All-Australian team (22 players) or the initial squad (44 players).
  • fetch_rising_star() - Returns Rising Star nominations and round-level performance stats.
